# ETEEAP Frontend

Minimal frontend setup for ETEEAP Project.

### Includes:
- React + Vite
- Tailwind CSS
- React Router DOM
- Axios

### How to run
1. Extract the zip
2. Navigate to frontend folder:
   ```bash
   cd eteeap/frontend
   npm install
   npm run dev
   ```
3. Open the link shown (usually http://localhost:5173)
